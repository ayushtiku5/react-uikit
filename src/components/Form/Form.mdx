# Form
### Easily create nice looking forms with different styles and layouts.

| Sub-component | Description |
|-|-|
| `Form.Fieldset` | The form fieldset. |
| `Form.Legend` | The form legend. |
| `Form.Controls` | The form controls. |

## Usage
<Playground>
  <Form>
    <Form.Fieldset>
      <Form.Legend>Form</Form.Legend>
      <Margin>
        <Input placeholder="Your name" />
      </Margin>
      <Margin>
        <Input placeholder="Your email" />
      </Margin>
      <Margin>
        <Input.Textarea placeholder="Your message" />
      </Margin>
      <Button type="submit" variant="primary">Submit</Button>
    </Form.Fieldset>
  </Form>
</Playground>

### Stacked form

    <Margin>

    </Margin>
    <Margin>
      <Input.Label htmlFor="select1">Select</Input.Label>

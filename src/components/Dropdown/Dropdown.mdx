# Dropdown
### Defines different styles for a toggleable dropdown.

| Sub-component | Description |
|-|-|
| `Dropdown.Container` | This is the container for the dropdown. |

## Usage
<Playground>
  <Dropdown.Container>
    <Button>Hover</Button>
    <Dropdown>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.
    </Dropdown>
  </Dropdown.Container>
  <Dropdown.Container>
    <Button>Click</Button>
    <Dropdown mode="click">
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.
    </Dropdown>
  </Dropdown.Container>
</Playground>

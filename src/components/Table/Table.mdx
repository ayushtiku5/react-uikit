# Table

> Work in progress

##### The `Table` component includes these sub-components:
| Sub-component | Description |
|-|-|
| `Table.Caption` | The caption for the table. |
| `Table.Head` | The table head `<thead>`. |
| `Table.Body` | The table body `<tbody>`. |
| `Table.Footer` | The table footer `<tfoot>`. |
| `Table.Row` | The table row `<tr>`. |
| `Table.Cell` | The table cell `<th>` or `<td>`. |

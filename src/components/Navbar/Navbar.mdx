# Navbar
### Create a navigation bar that can be used for your main site navigation.

| Sub-component | Description |
|-|-|
| `Navbar.Block` | The inner block for the navbar. |

## Usage

<Playground>
  <Navbar>
    <Navbar.Block>
      <Nav navbar>
        <Nav.Item active>
          <Link>Active</Link>
        </Nav.Item>
        <Nav.Item>
          <Link>Parent</Link>
          <Dropdown navbar>
            <Nav>
              <Nav.Item active>
                <Link>Active</Link>
              </Nav.Item>
              <Nav.Item>
                <Link>Item</Link>
              </Nav.Item>
              <Nav.Item>
                <Link>Item</Link>
              </Nav.Item>
            </Nav>
          </Dropdown>
        </Nav.Item>
        <Nav.Item>
          <Link>Item</Link>
        </Nav.Item>
      </Nav>
    </Navbar.Block>
  </Navbar>
</Playground>

# Nav
### Defines different styles for list navigations.

| Sub-component | Description |
|-|-|
| `Nav.Item` | The item for the nav. |
| `Nav.Header` | The header for the nav. |
| `Nav.Divider` | The divider for the nav. |

## Usage
<Playground>
  <Nav>
    <Nav.Item active>
      <Link>Active</Link>
    </Nav.Item>
    <Nav.Item>
      <Link>Item</Link>
    </Nav.Item>
    <Nav.Item>
      <Link>Item</Link>
    </Nav.Item>
  </Nav>
</Playground>

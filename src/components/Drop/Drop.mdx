# Drop
### Position any element in relation to another element.

| Sub-component | Description |
|-|-|
| `Drop.Container` | This is the container for the drop. |

## Usage
<Playground>
  <Drop.Container>
    <Button>Hover</Button>
    <Drop>
      <Card body>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.</Card>
    </Drop>
  </Drop.Container>
  <Drop.Container>
    <Button>Click</Button>
    <Drop mode="click">
      <Card body>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.</Card>
    </Drop>
  </Drop.Container>
</Playground>
